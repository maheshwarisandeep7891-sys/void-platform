## What is VOID?

VOID is a platform built exclusively for developers. No algorithm. No engagement bait. Just builders.

### 🖤 Dark Mode — Anonymous Posting
One click → you become `ghost_0x7f`. Zero link to your real identity. Ask anything without reputation damage.

### 🛒 Marketplace
Buy and sell unused developer resources:
- GitHub Copilot seats
- OpenAI API credits
- JetBrains licenses
- GPU access (rent by the hour)
- Side projects with revenue

Built-in escrow. No Stripe needed. Auto-refund if seller disappears.

### 📚 Knowledge Base
Stack Overflow done right:
- Anonymous questions
- "Still works" community verification
- Voting and accepted answers
- No hostile moderation

### 💰 Bounties
Post a problem with a reward. Community solves it. Escrow-backed payments.

### 🏆 Reputation System
`NEWCOMER → BUILDER → HACKER → ARCHITECT → LEGEND`

Exportable as a signed JSON credential for your resume.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js 15, TypeScript, Tailwind CSS v4, Framer Motion |
| Backend | Next.js API Routes, Prisma ORM |
| Database | PostgreSQL (Neon) |
| Auth | Custom JWT OAuth (GitHub) |
| Caching | Redis (Upstash) |
| Storage | Cloudflare R2 |
| Deployment | Vercel |

## Features

- ✅ Social feed (5 post types: Thread, Snippet, Drop, Bounty, Question)
- ✅ Monaco code editor with syntax highlighting
- ✅ Marketplace with internal escrow system
- ✅ Knowledge base with voting and answer acceptance
- ✅ Bounties with escrow-backed rewards
- ✅ Guilds organized by tech stack
- ✅ Hackathon rooms with live countdown
- ✅ Direct messages (polling-based)
- ✅ Real-time notifications
- ✅ Reputation system with credential export
- ✅ Anonymous posting (Dark Mode)
- ✅ Keyboard shortcuts (G+F, G+M, G+K, etc.)
- ✅ PWA manifest (installable)
- ✅ Rate limiting middleware
- ✅ SEO optimized (sitemap, robots.txt, OG images)
